# ShareIt - Multi Module Project.

Многомодульное приложение на Spring Boot, состоящее из двух основных модулей:

- **gateway** — шлюз, предоставляющий API для клиентов
- **server** — серверная часть, обрабатывающая бизнес-логику и работающая с базой данных

Проект предназначен для управления вещами, бронированием и запросами на вещи между пользователями.

## Архитектура

Проект использует архитектуру с централизованным сервером и клиентским шлюзом. Шлюз (gateway) перенаправляет запросы на
сервер (server), который взаимодействует с базой данных PostgreSQL.

## Технологии

- Java 21
- Spring Boot 3.3.2
- Maven
- PostgreSQL 16.1
- Docker / Docker Compose
- Lombok
- JUnit 5

## Запуск проекта

1. Убедитесь, что у вас установлен Docker и Docker Compose
2. Выполните команду:
   ```bash
   docker-compose up --build
   ```
3. После запуска:
    - Шлюз будет доступен на порту 8080
    - Сервер — на порту 9090
    - База данных — на порту 6541

## Структура проекта

- `gateway/` — клиентский шлюз
- `server/` — серверная часть с REST API и бизнес-логикой
- `docker-compose.yaml` — конфигурация для запуска всех сервисов в контейнерах
- `pom.xml` — основной файл сборки Maven с настройками для проверки кода (Checkstyle, SpotBugs, JaCoCo)
